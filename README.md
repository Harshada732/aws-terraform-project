AWS EC2 Infrastructure using Terraform

Project Description

This project provisions AWS infrastructure using Terraform.

Resources Created

- VPC
- Public Subnet
- Internet Gateway
- Route Table
- Security Group
- EC2 Instance

Monitoring

- Basic monitoring using AWS CloudWatch

Technologies Used

- AWS
- Terraform
- Git
- GitHub

Security Group Rules

- SSH (Port 22)
- HTTP (Port 80)

Deployment Steps

1. terraform init
2. terraform plan
3. terraform apply
